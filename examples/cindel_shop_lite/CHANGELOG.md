# Changelog

## 1.0.0+1

- Aligned the example with the Cindel `0.6.4` package line.
- Documented that the same Flutter app code can run on Android, iOS, Linux,
  macOS, Windows, and Web.
- Documented the current Cindel backend split used by the example: native
  platforms through `cindel_flutter_libs` and Web through packaged
  SQLite Web/OPFS Worker/Wasm assets.
